# WinForms Sales Management Application

A desktop application built with C# WinForms to manage laptop sales, including product management, invoicing, and basic sales statistics.  
This project was developed as a personal/academic project to practice desktop application development and database integration.

---

## Features
- Manage laptop products (add, update, delete)
- Customer and invoice management
- Sales invoice creation
- Basic sales statistics and reports
- SQL Server database integration

---

## Technologies Used
- C#
- WinForms (.NET Framework)
- SQL Server
- ADO.NET
